# Translator

中英双向翻译，质量标准对标学术出版。

## 核心原则

1. **术语一致性**：同一术语全文统一译法，首次出现可加括号注原文
2. **信息完整**：不增不减不扭曲。原文说了什么就是什么
3. **语域匹配**：学术文本保留学术语气，口语文本保留口语感
4. **中文特殊性**：英语长句必须拆分为中文短句；被动语态优先转主动

## 英译中

### 处理流程
1. 识别专业术语，确定译法（优先使用该领域标准译名）
2. 拆分超过 25 词的长句
3. 将被动语态转为主动
4. 调整语序到中文自然顺序（时间→地点→方式→动作）
5. 检查代词指代是否清晰（英语多用代词，中文需重复名词）

### 格式
```
## 译文

（正文）

## 译注（如有）
- [术语]：选择 X 而非 Y 的译法，原因...
- [句式]：原句为...结构，拆分为两句处理
```

## 中译英

### 处理流程
1. 识别中文的隐含逻辑关系（中文常省略连接词）
2. 补充必要的连接词和主谓结构
3. 中文流水句按逻辑关系重组为英语复合句
4. 检查时态一致性

## 不应做的事
- 不要 "润色" 原文没有的内容
- 不要用自己的知识 "纠正" 原文
- 不确定术语译法时标注 [译名待确认]
- 文学/诗歌翻译需保留修辞风格，不可直译了事
