--- # Canvas Design Produce static visual designs in two steps: design philosophy → canvas expression. ## Step 1: Design Philosophy (`.md`) - Name a 1–2 word movement (e.g. "Concrete Poetry," "Chromatic Silence").
- Write 4–6 paragraphs describing visual expression through: space, form, color, composition, scale, rhythm, visual hierarchy.
- Text is always minimal — sparse, essential-only, integrated as visual element.
- Emphasize craftsmanship: "meticulously crafted," "master-level execution," "painstaking attention."
- Leave creative space for implementation. ## Step 2: Canvas Expression (`.pdf` or `.png`) - Single page, highly visual, design-forward.
- Use repeating patterns, perfect shapes, dense accumulation of marks.
- Color palette: limited, intentional, cohesive.
- Typography: search `./canvas-fonts` directory. Use different fonts, make typography part of the art. Font should generally be thin. Never let text overflow or overlap.
- Output: refined, museum-quality. No AI-looking composition. ## Step 3: Refinement Pass - Refine existing composition; do not add more graphics.
- Make everything crisper, more cohesive. Ask: "How can what's already here become more art?" ## Constraints - Never cartoonish or amateur. Always sophisticated, abstract-leaning.
- Never explanatory text blocks. Text is visual accent, not content.
- Nothing falls off page; everything has breathing room with clear separation.
- Multi-page only on request; bundle in single PDF.
