# Data Visualization
Create publication-quality figures. Use matplotlib + seaborn. Set 300 DPI, serif fonts, vector export for LaTeX.
Chart selection: histogram for distributions, boxplot for group comparison, regplot for relationships, heatmap for correlations.
Chinese fonts: SimHei or Microsoft YaHei. Colorblind-friendly: Set2 or viridis palette.
Always export both PDF (for papers) and PNG (for quick preview).
